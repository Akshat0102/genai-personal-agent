# GenAI Personal Agent

GenAI Personal Agent is a multi-purpose personal AI assistant designed to help with a variety of tasks, such as planning, learning, and providing useful information. This project leverages OpenAI's GPT models and integrates tools to enhance its functionality.

## Features
- **AI-Powered Assistance**: Uses OpenAI's GPT models to provide intelligent responses.
- **Custom Tools**: Includes tools for weather updates, time retrieval, calculations, and text summarization.
- **RESTful API**: Built with FastAPI for easy integration.
